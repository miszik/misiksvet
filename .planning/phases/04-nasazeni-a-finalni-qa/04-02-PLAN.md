**Kroky které musí udělat Míša (v tomto pořadí):**

---

### Krok 1: GitHub — vytvoření repozitáře a push kódu

1. Otevřít github.com a přihlásit se jako `misaz`
2. Kliknout na **+** → **New repository**
3. Nastavení:
   - **Name:** `misiksvet`
   - **Visibility:** Public (nutné pro GitHub Pages na free účtu)
   - **Initialize this repository with:** NEZAŠKRTÁVAT nic (kód už máme)
4. Kliknout **Create repository**
5. V terminálu v root složce projektu (`web_misiksvet/`) spustit:
   ```bash
   git remote add origin https://github.com/misaz/misiksvet.git
   git branch -M main
   git push -u origin main
   ```
   GitHub může vyžádat přihlášení — zadat GitHub heslo nebo token.

---

### Krok 2: GitHub Pages aktivace

1. Na stránce repozitáře jít do **Settings** → **Pages**
2. Pod "Source" vybrat: **Deploy from a branch**
3. Branch: **main**, Folder: **/ (root)**
4. Kliknout **Save**
5. Počkat 2–3 minuty (GitHub build trvá chvíli)
6. URL se zobrazí nahoře: `https://misaz.github.io/misiksvet/`
7. Otevřít URL v prohlížeči, udělat hard refresh (Ctrl+Shift+R)

**POZOR:** V Settings/Pages se může zobrazit žluté varování o custom doméně — to je normální, ignorovat.

---

### Krok 3: EmailJS konfigurace (PŘED prvním odesláním objednávky)

Jít na `https://dashboard.emailjs.com/`

**A) Allowed Origins (bezpečnost — kritické):**
1. Account → Security → Allowed Origins
2. Přidat: `https://misaz.github.io`
3. Formát MUSÍ být přesně takto (pouze origin, bez `/misiksvet/` cesty)
4. Uložit

**B) Šablona pro zákazníka (potvrzení objednávky):**
1. Email Templates → Create New Template
2. Name: `Potvrzeni objednavky`
3. Subject: `Potvrzení objednávky — Mišik svět`
4. Body šablony — použít tyto proměnné (EmailJS syntaxe):
   - `{{full_name}}` — jméno zákazníka
   - `{{email}}` — e-mail zákazníka
   - `{{delivery}}` — způsob doručení
   - `{{pickup_point}}` — výdejní místo (pokud relevantní)
   - `{{home_address}}` — adresa domů (pokud relevantní)
   - `{{cart_summary}}` — shrnutí košíku (více řádků)
   - `{{total}}` — celková cena
   - `{{note}}` — poznámka
5. Uložit → zkopírovat **Template ID** (formát: `template_xxxxxxx`)

**C) Šablona pro Míšu (notifikace s detaily):**
1. Email Templates → Create New Template
2. Name: `Notifikace objednavky`
3. Subject: `Nová objednávka — {{full_name}}`
4. Body: použít stejné proměnné jako zákazník + navíc `{{phone}}`
5. To: nastavit na Míšin e-mail
6. Uložit → zkopírovat **Template ID**

**D) Email Service:**
1. Email Services → Add New Service → vybrat Gmail nebo jiný
2. Propojit s e-mailem
3. Zkopírovat **Service ID** (formát: `service_xxxxxxx`)

**E) Public Key:**
1. Account → General → Public Key
2. Zkopírovat klíč

**F) Vložit klíče do kódu:**
- `index.html` řádek 21: nahradit `YOUR_EMAILJS_PUBLIC_KEY` skutečným Public Key
- `js/order.js` řádek 9: nahradit `YOUR_SERVICE_ID` skutečným Service ID
- `js/order.js` řádek 10: nahradit `YOUR_CUSTOMER_TEMPLATE_ID` ID šablony zákazníka
- `js/order.js` řádek 11: nahradit `YOUR_OWNER_TEMPLATE_ID` ID šablony pro Míšu
- Uložit soubory, spustit `git add -A && git commit -m "feat: EmailJS klíče" && git push`
